
# Linux Installation


## Install OpenCV
```
sudo apt-get install libopencv-dev
```


## Install Python3 Modules
```
$ sudo pip3 install opencv-python
$ sudo pip3 install tensorflow
$ sudo pip3 install numpy
$ sudo pip3 install pandas
$ sudo pip3 install exifread
```
